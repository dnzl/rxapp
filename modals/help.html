This is a web-based viewer and encrypter for DICOM files.

It uses DWV (DICOM Web Viewer), which is an open source zero footprint medical image viewer.

Upon saving all files are encrypted client-side using CryptoJS, then saved in an AWS-like environment.

Click Demo below to load a demo file and try this thing!

This web-app is just an exercise, full of bugs and by no means intended to be used for realsies.
Beware! All uploaded files can be deleted at any time!
